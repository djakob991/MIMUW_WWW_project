# MIMUW_WWW_project
Aplikacja stworzona w ramach przedmiotu WWW na MIMUW.

## Aby uruchomić lokalnie:
```
npm install
npm run build-frontend
npm run createdb
npm run build     // Alternatywnie, 
npm run start     // npm run dev
```

## Testowanie
```
npm test
```

## Czyśczenie
```
npm run clean 
npm run clean-frontend
```
